## 🚀 How to Run

1. 📥 Clone the repository:
   ```bash
   git clone https://github.com/Ayushkumar418/Python-Programs.git
   cd Python-Programs
   ```

2. ▶️ Run a program:
   ```bash
   python filename.py
   ```

   Replace `filename.py` with any file name you want to run.
